overwatch

o overwatch é um jogo de fps competitivo no qual é composto por herois, onde são 3 classes entre elas tank,dps(dano) e suporte

Principais pontos

  • ter uma visão de jogo
  • saber jogar com pelo menos 1 heroi de cada classe
  • saber ultilizar as ultis
  • ter comunicação com o time
  • ter bons perififerico para jogar o game
